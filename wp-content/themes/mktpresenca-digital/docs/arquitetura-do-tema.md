1. Habilitar imagem destacada para todos os CPTs necessários.

2. Criar tamanhos oficiais:
   - thumbnail padrão
   - P / mobile
   - M / tablet
   - G / desktop
   - Hero desktop
   - Hero mobile

3. Definir recomendações no painel:
   - Imagem destacada: 900x600 px • WebP • até 120 KB
   - Hero desktop: 1920x1080 px • WebP • até 250 KB
   - Hero mobile: 768x432 px • WebP • até 90 KB

4. Usar esses tamanhos no front com:
   - picture
   - srcset
   - sizes
   - loading lazy quando não for imagem principal
   - fetchpriority high apenas na hero principal

5. Depois tratar conversão WebP.
